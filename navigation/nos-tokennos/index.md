# nOS Token(NOS)

NOS（NOS Universal Smart Contract）是一个新的、透明的互联网虚拟操作系统。它由NEO智能经济提供技术支持。nOS 团队由具有区块链和智能合约开发历史的经验丰富的开发人员和研究人员组成。扎根于锡安市，团队的许多成员都在（并为）NEO 区块链上进行了开发。
