# BlockTrade(BTT)

Blocktrade是一个高性能的,即将成为MiFID II监管的虚拟币交易平台。Blocktrade作为顶级交易平台，提供加密币，加密交易指数和其他区块链相关金融工具。我们将基于区块链的技术和加密币引入统一且互联的市场。Blocktrade受瑞士保护，安全性高，功能齐全，同时由性能驱动，成交量高，具有市场深度，符合法律法规。
