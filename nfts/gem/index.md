# Gem

在 Gem，我们正在构建最好的 NFT 聚合器，让您可以从 A 到 XYZ 访问整个 NFT 市场。
在所有市场中找到最优惠的价格
在一次交易中购买多个 NFT
扫地收集地板
使用任何代币（或多个代币）付款
与直接在市场上购买相比，节省大量天然气我们的使命是提供一流的聚合，从而产生市场上最优惠的价格、最大的 NFT 供应和最完整的数据 - 从以太坊区块链开始。
