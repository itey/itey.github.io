# Lokian Monsters

一款集换式卡牌游戏 dapp，在 Solidity 区块链中以 NFT、分享、战斗和繁殖神话生物为特色。
像 Polygon、BSC、Fantom、Avalanche、Astar 等
功能包括
通过竞技场决斗赚取 ERC20 Lokian 代币
通过代币专区购买 ERC1155 NFT Lokian 物品
卖掉你的怪物并赚取原生代币
用原生代币购买怪物
培育你的怪物以创造后代
与您的朋友和爱人分享您的怪物！
