# Farmers World

Farmers World 是第一个在 NFTs 平台上运行的农场游戏。为自己挑选合适的工具，开发各种资源，购买土地来建造巨大的农场，并享受在农民世界生态系统中工作的农民的迷人经历。

最大的 NFT 游戏社区之一，在全球拥有超过 150000 名玩家。
有任何问题或只是想与其他探险家分享您的经验？我们的 Telegram 和 Discord 社区活跃而友好——您可以随时了解更多信息并在那里聊天。
